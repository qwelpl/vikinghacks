# Warden
*An AI Extension of the traditional lockdown browser*
##  Main Features
**Lock tabs**
- Blocks all the websites and tabs and whitelists the ones you need to use to complete your work.
At the beginning, you tell it what work you need to do and what tabs you need, along with a reason for each
Only opens up after you have shown sufficient proof that you have finished all work. 
Needs sufficient proof (such as quiz accuracy, completing work); Needs to check the
assignment prompt to make sure you actually did something. If you try to make an invalid excuse,  
uses an algorithm to make it detect whether what you are saying is realistic; Forces you to complete what 
you want to 

**Unlock tabs**
- You make a valid excuse and lets you use the specific tab you need for a specified period of time
- The AI judges if you really need to access the tabs (it is lenient)
- Option to make another person give you approval to access the tab
- Whitelist (pages that are always allowed)
- Administrator password to end lock down
- Optional break sessions feature every 25 min for 5 mins (by default; you can change in the beginning)
- Employs an algorithm to detect whether it uses
